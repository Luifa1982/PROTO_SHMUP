# PROTO_SHMUP
Prototipo de shot'em up (matamarcianos) en Pygame con enfoque a OOP

TP integral de la materia Programación Orientada a Objetos dictada por el Prof. Javier Castrillo
para la Universidad Nacional de Pilar, Comisión 4 1er Cuatrimestre 2026

PROTO_SHMUP es un videojuego estilo "shooter vertical" o tambíen conocido como shoot'em up, shmup o matamarcianos.
El juego consiste en llegar al final del mismo, sobreviviendo el ataque de distintos tipos de enemigos.
Controlamos una nave en las 2 direcciones de movimiento, arriba/abajo, izquierda/derecha. Y con los botones podemos disparar un laser o tirar una bomba.
